# Echo AI: 잔존 청력 최적화 및 청능 재활을 위한 혁신 솔루션 개발 계획

본 계획은 청각 장애(특히 노인성 난청 및 부분 실청) 환자의 **'잔존 청력 활용 극대화'**와 **'인지 부하 감소'**를 핵심 과학적 논리로 채택한 단계별 개발 로드맵입니다.

---

## 🔬 과학적 근거 (Scientific Logic)

1.  **다감각 통합 (Multimodal Integration):** 뇌는 시각 정보와 청각 정보를 동시에 처리할 때 정보의 신뢰도를 높입니다. 아버님처럼 유추에 의존하는 상황은 청각 정보의 손실을 뇌의 인지적 노력을 통해 메우는 과정이며, 이는 상당한 피로를 유발합니다. 실시간 시각 자막은 이 '인지 비용'을 낮추는 핵심 역할을 합니다.
2.  **청능 재활의 4단계 (Erber's 4 Stages):** 학술적으로 인정받는 청능 재활 프로세스(감지 -> 변별 -> 확인 -> 이해)를 앱 내 훈련 모듈에 이식합니다.
3.  **음성 강화 알고리즘 (Speech Augmentation):** 특정 주파수 대역(주로 고주파수)의 소리 손실을 보충하기 위한 주파수 압축 및 증폭 기술을 적용합니다.
4.  **신경 가소성 (Neuroplasticity):** 지속적인 청각 자극과 훈련을 통해 뇌의 청각 피질이 새로운 소리 패턴에 적응하도록 유도합니다.

---

## 🛠 단계별 개발 로드맵

### Phase 1: 기반 구축 (Real-time Vision) - [현재 진행 완료]
*   **핵심 기술:** Web Speech API 기반 실시간 자막 모듈.
*   **과학적 목표:** 대화 시 '유추' 단계를 '확인' 단계로 전환.
*   **기여:** 실시간 중간 인식 결과 표시를 통해 지연 시간(Latency)을 최소화하여 대화의 맥락 유지.

### Phase 2: 청각 강화 및 소음 제어 (Audio Augmentation)
*   **기술적 구현:** 
    *   **고대역 주파수 부스팅:** 난청 환자들이 놓치기 쉬운 마찰음(ㅅ, ㅎ 등) 대역 강화.
    *   **AI 소음 억제:** Deep Noise Suppression 기술을 활용하여 배경 소음과 목소리 분리.
*   **과학적 목표:** SNR(Signal-to-Noise Ratio, 신호 대 잡음비)을 개선하여 음성 변별력 향상.

### Phase 3: 환경 지능 서비스 (Environmental Intelligence)
*   **기술적 구현:** 
    *   **Sound Radar:** 자동차 경적, 초인종, 가전제품 이동 알림 등 생활 밀착형 소리 감지.
    *   **위치 기반 알림:** 위험 상황 발생 시 시각/진동 피드백.
*   **과학적 목표:** 외부 자극에 대한 인지적 반응 속도 개선 및 생활 안전 확보.

### Phase 4: 신경 재활 시스템 (Neuro-Rehab Modules)
*   **기술적 구현:** 
    *   **어음 변별 게임:** 유사 발음(사/차/파)을 구분하는 개인 맞춤형 훈련.
    *   **단계별 난이도 조절:** 사용자의 수행 능력에 따른 Erber 4단계 커리큘럼 자동 생성.
*   **과학적 목표:** 뇌의 청각 처리 효율성 증대 및 인지 저하 예방.

## 📋 사용자 허가가 필요한 주요 마일스톤

1.  **AI 엔진 업그레이드:** Whisper/Gemini 2.0 기반의 초정확 STT 모델 도입 시 (성능 극대화).
2.  **커스텀 오디오 프로필:** 아버님의 청력 검사 결과(Audiogram) 정보를 앱에 입력하여 개인화된 주파수 튜닝 서비스를 시작할 때.
3.  **데이터 프라이버시 설정:** 생활 소리 감지를 위해 앱이 상시 마이크를 켜둘 필요가 있는 "Guardian Mode" 활성화 시.
